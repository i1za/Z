# Eatemad ERP - Project Guidelines

## Tech Stack
- Frontend: React.js (Vite)
- Language: JavaScript (JSX)
- Styling: Custom CSS (index.css) / Dark Mode Theme
- State Management: Context API / Redux (if used)

## Design System & UI Rules
- Always prioritize a luxurious, professional "Dark Mode" aesthetic.
- Ensure full bilingual support (Arabic primarily, English secondary) with proper RTL (Right-to-Left) rendering for Arabic.
- Respect existing component structures and do not overwrite the light mode styles.

## Git Workflow
- Always ask for permission before staging or committing files.
- Keep commits descriptive and small.

## General Instructions
- Do not modify existing core code without explicit permission.
- Write clean, modular, and reusable components.
- Think step-by-step before implementing new modules (HR, Accounting, CRM, etc.).
